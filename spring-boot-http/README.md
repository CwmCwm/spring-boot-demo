### spring-boot-http的目录（以maven项目目录为例）
- resources/static/ 目录是spring-boot http服务默认的静态资源目录（你可以在浏览器能访问哪些静态资源，url该怎么写）
- resources/application.properties 文件是spring-boot默认加载的配置文件（可以命令行指定别的文件）
